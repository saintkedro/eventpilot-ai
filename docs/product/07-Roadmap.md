# Roadmap

WhatsApp-first build order.

---

## Sprint 1 — WhatsApp + Conversation Core

- WhatsApp Business API integration
- Webhook handling, message in/out
- User/org linking via phone
- Basic conversational AI (event intake)
- Event record creation from chat

**Exit criteria:** User messages EventPilot → event is created in backend.

---

## Sprint 2 — Event Lifecycle in Chat

- Publish event, update via conversation
- Auto-generate public event page
- RSVP collection (WhatsApp + web link)
- Status queries ("Who hasn't responded?")

**Exit criteria:** Organizer runs invite + RSVP flow entirely in WhatsApp.

---

## Sprint 3 — Tickets, Reminders, Event Day

- Free/paid ticketing, QR codes
- Reminder scheduling via chat
- Check-in (QR + manual)
- Post-event summary in chat

**Exit criteria:** Full MVP loop complete without dashboard.

---

## Sprint 4 — Polish + Optional Dashboard

- Human tone refinement, proactive nudges
- CSV export via chat link
- Minimal dashboard (guest list, analytics) for power users
- Auth recovery, team access (if time)

**Exit criteria:** MVP validated with real organizers.
