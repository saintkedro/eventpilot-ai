# Vision

## The Problem

Event planning today is fragmented.

Organizers use WhatsApp groups, spreadsheets, email, ticketing links, and phone calls — but nothing connects them into one flow.

WhatsApp is where coordination happens. It is not where events are actually managed.

---

## The Opportunity

WhatsApp is universal. AI is now good enough to understand natural conversation.

Instead of asking people to leave WhatsApp and learn event software, EventPilot meets them in the chat they already use every day.

Example:

> "I'm planning a 150-person church conference in Abuja on the last Saturday of August. I need RSVPs and reminders."

EventPilot responds like a human coordinator:

- Clarifies missing details through conversation
- Creates the event
- Generates a shareable event page
- Helps invite guests
- Tracks RSVPs
- Sends reminders
- Supports check-in on event day
- Delivers a summary after the event

All from WhatsApp.

---

## Product Vision

EventPilot is an event management platform on WhatsApp.

Organizers create, manage, and run events through simple, human conversation.

The web exists only where WhatsApp cannot — shareable links, payments, and lightweight guest flows.

There is no requirement to use a dashboard. The dashboard is optional for power users who want a visual overview.

---

## What Success Looks Like

- A birthday party organizer never opens a website
- A church admin runs registration entirely from chat
- A conference organizer checks RSVPs by asking "Who hasn't responded yet?"
- Guests receive clear, timely messages without chaos in group chats
- Event day feels organized, not stressful
